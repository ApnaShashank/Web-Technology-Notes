- What is HTML (HyperText Markup Language)? - HTML is the language for publishing web pages on the WWW (World-Wide Web, or World-Wide Wait?). - HTML is a Document Description Language (aka Document Markup Language). HTML is the standard markup language for documents designed to be displayed in a web browser. HTML is NOT a programming language like C/C++/C#/Java, which is used to implement programming algorithm. - An HTML document is a text document, and it is human-readable. - Today, HTML works together with CSS (Cascading Style Sheets) (for layout) and JavaScript (for programming).
- Line 1: HTML ka full form HyperText Markup Language hai. - Line 2: Yeh web pages banane ki language hai WWW ke liye (World Wide Web). - Line 3: Yeh ek markup language hai, documents ko describe karta hai, programming language nahi hai jaise C ya Java. - Line 4: HTML file ek text file hoti hai, jo aasani se padhi ja sakti hai. - Line 5: Aaj kal HTML ke saath CSS (design ke liye) aur JS (programming ke liye) kaam karta hai.
